What’s inside l8r How the dependancies works. This tool generate representation of binaries using YAML meta language. Three type of assets can be generated. The header file File containing all the resources that can be loaded for a particular binary. example: file: binary_test arch: AMD64 metas: - something - something_else artefacts: - name: main filename: main.yaml symbol_name: main - name: call filename: call.yaml symbol_name: main The function file File containing all the informations that belongs to a particular symbol within a binary. »
