The VF2 Algorithm

Algorithm The VF algorithm is an improvement of the VF algorithm. The application of this algorithm is to detect isomorphism and subgraph isomorphism. The improvement of VF2 is the requirement of a lower complexity (from O(N^2) to O(N)) How does it works This algorithm implements a naive detection of isomorphism. Function Level Isomorphism Detection

Idea The idea behind function level detection is simple. In MalEx it means that every functions symbol that can be extracted from a binary will be compared.
